# 🏥 StayHealthy - Plateforme de santé

## Nom du projet
**StayHealthy Frontend Application**

## Description
Plateforme web permettant aux patients de prendre rendez-vous avec des médecins en ligne, avec gestion des avis, profil utilisateur et notifications.

## Technologies
- React.js
- CSS3
- Axios pour les appels API
- React Router v6

## Installation

```bash
git clone https://github.com/ju-info-19/stayhealthy-frontend.git
cd stayhealthy-frontend
npm install
npm start
